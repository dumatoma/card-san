## 1. 产品概述

### 1.1 产品愿景

Card-San（卡桑）是面向日本本地生活服务门店（美容院、美甲店、健身房、餐饮等）的一体化数字会员管理平台。

- **消费者（C端）**：用手机统一管理多家店铺的电子会员卡，使用优惠券、预约服务、与店铺聊天
- **店主/店员（B端）**：顾客管理（CRM）、营销通知、菜单与营业时间、预约管理、会员卡设计、订阅与支付

### 1.2 目标用户

| 用户类型 | 说明 |
|---------|------|
| 消费者 | 日本在住，希望用手机统一管理多家店会员卡的用户 |
| 店铺老板 | 美容、美甲、健身、餐饮等本地门店经营者 |
| 店铺员工 | 负责顾客接待、预约管理的工作人员 |
| 系统管理员 | Card-San 平台整体的运营管理者（使用 admin） |

### 1.3 仓库结构

| 仓库 | 内容 |
|-----|------|
| `card-san` | 前端（uni-app 多端 + admin 管理后台） |
| `api.card-san.jp` | 后端 HTTP API（Laravel 8，PHP） |

### 3.3 jpRegister — 新商户注册

**AppID**: `__UNI__ED993F2`  
**目标平台**: H5（浏览器）+ 各移动平台

#### 功能与流程

三步注册流程：

1. **第 1 步 / 共 3 步** — 输入邮箱地址并发送验证邮件
2. **第 2 步 / 共 3 步** — 输入 6 位邮箱验证码（支持重新发送）
3. **第 3 步 / 共 3 步** — 填写姓名、店铺名，同意用户协议
4. **完成页** — 显示账号 ID，引导下载 CardSan管理 App

**附加页面**：用户协议、隐私政策展示页

### 3.4 outsiteAppointment — 外链轻量预约

**AppID**: `__UNI__26F5B98`  
**目标平台**: H5（从店铺外部链接跳转进入）

#### 功能与流程

5 个页面构成的轻量预约完整流程：

1. **菜单列表** — 按分类展示服务项目，支持选择与收藏
2. **菜单详情** — 完整菜单（含时长与价格）
3. **日期/时间/员工选择** — 日语日历 + 指定担当员工
4. **预约内容确认** — 来店时间、菜单、担当、指名料
5. **预约完成** — 预约编号与详细信息展示

**组件**：自定义日语日历、周视图、拖拽排序

## 4. 后端 API 规格概述

**生产域名**: `https://api.card-san.jp`  
**框架**: Laravel 8（PHP）

### API 路径设计

| 路径前缀 | 对应端 |
|---------|-------|
| `/api/member/*` | 会员端（jpcard、outsiteAppointment） |
| `/api/shop/*` | 门店管理端（jpshop、jpRegister） |
| `/adminapi/*` | 系统管理（admin） |
| `/api/config` | 公共配置获取 |

### 认证方式

- 请求头携带 `token` + `uuid`
- Content-Type: `application/json;charset=UTF-8`

### 实时通信

- WebSocket 连接地址：`ws://47.245.1.2:21000`
- 连接成功后绑定 `client_id`（`/api/member/bind` 或 `/api/shop/bind`）
